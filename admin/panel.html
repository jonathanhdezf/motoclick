¡Paso importante! Para guardar nuevas contraseñas, debes habilitar la base de datos.

1. Entra a tu proyecto en Supabase.
2. Ve al SQL Editor.
3. Copia y ejecuta este código exactamente así:
ALTER TABLE users ADD COLUMN IF NOT EXISTS pin TEXT;
4. Luego, cierra esta ventana e inténtalo de nuevo.
